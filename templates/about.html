Api Franklin позволяет получить информацию о курсах и анонимных результатах прохождений тестов (результаты отдельно взятого пользователя получить невозможно!).

Информация о курсах

/api/courses?side_id=1&category_id=14&course_id=11

НЕ рекомендуется использовать сразу несколько параметров

  • side_id: уникальный индентификатор каждого направления
  • category_id: уникальный индентификатор каждой категории
  • course_id: уникальный индентификатор каждого курса

Результаты тестирований

/api/results?test_id=1&result_id=15

НЕ рекомендуется использовать сразу несколько параметров

  • test_id: уникальный индентификатор каждого теста
  • result_id: уникальный индентификатор каждого результата
